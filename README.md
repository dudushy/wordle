# [wordle](https://dudushy.github.io/wordle/)
**wordle** is a web game about finding the correct word within the certain amount of tries that you are given to do so.
---

## Prerequisites:
- [NodeJS](https://nodejs.org/)

## How to install:
1. Install Packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    ng serve
    ```
